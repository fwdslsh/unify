Slot Injection in Include Elements

What is Slot Injection?

Slot injection allows you to pass custom content to specific slots within an included component. This enables component reusability while maintaining flexibility for customization.

How It Works

  1. Define slots in your component using data-slot="name" attributes
  2. Include the component using the <include> element
  3. Provide content for slots by adding child elements with matching data-slot attributes
  4. The build process replaces slot targets with the provided content

Example Code

Component Definition

<nav class="u-nav">
  <div data-slot="brand">Default Brand</div>
  <ul data-slot="links">
    <li><a href="/">Home</a></li>
  </ul>
  <div data-slot="actions">
    <button>Sign In</button>
  </div>
</nav>

Using the Component with Slots

<include src="/_includes/nav.html">
  <a data-slot="brand" href="/">MyBrand</a>
  
  <ul data-slot="links">
    <li><a href="/docs/">Docs</a></li>
    <li><a href="/blog/">Blog</a></li>
  </ul>
  
  <div data-slot="actions">
    <a href="/start/" class="btn">Get Started</a>
  </div>
</include>

Benefits

  • Reusability: Define components once, use them everywhere with different content
  • Flexibility: Customize specific parts of a component without modifying the original
  • Fallback Content: Components can provide default content for unprovided slots
  • Build-time Resolution: All slot injection happens at build time, no runtime overhead
  • Clean Output: The data-slot attributes are removed from the final HTML
