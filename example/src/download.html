Usage

Once installed, you can build and serve your site with a single command:

                
unify build         # Build from src/ to dist/
unify serve         # Start dev server with live reload
unify watch         # Watch for changes and rebuild
For advanced options, see the CLI Reference.

Supported Platforms

  • Bun (fastest, native APIs)
  • Linux/MacOS/Windows (compiled binary)
  • Node.js 14+ (native ESM support)
