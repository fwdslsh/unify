Important Notes

Template vs Regular Elements

  • <template data-target="..."> - Content is hidden until processed, good for progressive enhancement
  • <div data-target="..."> - Content is visible in source, maintains semantic HTML
  • Both approaches work equally well for slot injection

Processing Rules

  • The data-import and data-target attributes are removed from final output
  • <slot> and <template> elements are replaced with their content
  • Content without data-target goes to the unnamed/default slot
  • If no content is provided for a slot, the slot's default content is used
