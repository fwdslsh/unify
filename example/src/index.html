Quick Install

One-Line Installation


# Install Bun (required runtime)
curl -fsSL https://bun.sh/install | bash

# Install unify globally
bun add -g @fwdslsh/unify

Then just run unify build to build your site or unify serve for live development!

Core Commands

Build

unify build
Generate your static site

Serve

unify serve
Development server with live reload

Watch

unify watch
Watch for changes and rebuild

Simple Example

HTML with Includes

<!-- src/index.html -->
<h1>Welcome!</h1>

  <!--#include virtual="/_includes/header.html" -->
  <main><slot></slot></main>
  <!--#include virtual="/_includes/footer.html" -->

</html>
