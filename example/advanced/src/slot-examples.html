Template Data-Slot Content

The sidebar navigation above uses <template data-slot="sidebar">, so it's hidden when viewing this file directly in a browser.

Element Data-Slot Content

The alert box above uses <aside data-slot="alert">, so it's visible when viewing this file directly in a browser.

When to Use Each Approach

<template data-slot="name">
Use for layout-specific content like navigation, meta tags, or content that only makes sense within the compiled layout.
<element data-slot="name">
Use for content that should be visible and meaningful even when viewing the raw page file directly.
