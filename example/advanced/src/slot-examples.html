Template Slot Content

The sidebar navigation above uses <template slot="sidebar">, so it's hidden when viewing this file directly in a browser.

Element Slot Content

The alert box above uses <aside slot="alert">, so it's visible when viewing this file directly in a browser.

When to Use Each Approach

<template slot="name">
Use for layout-specific content like navigation, meta tags, or content that only makes sense within the compiled layout.
<element slot="name">
Use for content that should be visible and meaningful even when viewing the raw page file directly.
